In the latest version of PORTOID, localization is no longer supported. However, the localized version created by [François Hernandez](https://github.com/francoishernandez) is still available as a [separate branch](https://github.com/Mrmarc-bit/portoid/tree/localization-(Portoid-0.4.2)). This branch will not receive further updates.
